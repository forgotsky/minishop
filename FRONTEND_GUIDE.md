# Frontend Guide (uni-app)

## Code Location
- Pages: `uni-app/pages/*`
- Data/logic: `uni-app/data/*`
- Utilities: `uni-app/utils/*`
- App shell: `uni-app/App.vue`, `uni-app/main.js`, `uni-app/pages.json`
- App config: `uni-app/manifest.json`

## Local Run (H5)
```bash
cd uni-app
npm install
npm run dev:h5
```

## Configure API URL
Default backend is `http://localhost:8000`.
To override in runtime:
```js
uni.setStorageSync('API_BASE_URL', 'https://api.yourdomain.com')
```

## Build (H5)
```bash
cd uni-app
npm install
npm run build:h5
```
Output goes to `uni-app/dist/build/h5`.

## Mini-App Build (HBuilderX)
1. Open `uni-app/` in HBuilderX
2. Set AppID in `manifest.json` for target platform
3. Run: 发行 -> 小程序 -> 选择平台（微信/抖音/支付宝）
4. Use platform devtools to预览/上传

## Platform Notes
- WeChat: requires `project.config.json` auto-generated by HBuilderX
- Douyin: requires Toutiao DevTools
- Alipay: requires Alipay Mini Program Studio

HBuilderX handles per-platform build output.
